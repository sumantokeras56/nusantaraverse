# Checklist Final Nusantaraverse untuk LIDM

## A. Produk Web App

- [ ] Landing page selesai.
- [ ] Login siswa/guru tersedia.
- [ ] Dashboard siswa selesai.
- [ ] Peta eksplorasi tampil.
- [ ] Marker lokasi bisa diklik.
- [ ] Detail misi tersedia.
- [ ] AI Tutor berjalan.
- [ ] Simulasi keputusan berjalan.
- [ ] Skor dampak tampil.
- [ ] XP dan badge tampil.
- [ ] Leaderboard tersedia.
- [ ] Dashboard guru tersedia.
- [ ] Grafik analitik tampil.
- [ ] Prototype virtual tersedia.
- [ ] Web berhasil di-deploy.

## B. Proposal

- [ ] Cover.
- [ ] Lembar pengesahan.
- [ ] Abstrak 150–200 kata.
- [ ] Latar belakang.
- [ ] Tujuan dan manfaat.
- [ ] Metode pengembangan.
- [ ] Analisis fungsional.
- [ ] Desain produk.
- [ ] Implementasi dan validasi.
- [ ] Tautan video.
- [ ] Daftar pustaka.
- [ ] Lampiran AI.
- [ ] Similarity report maksimal 25%.

## C. Video Demo

- [ ] Durasi maksimal 3 menit.
- [ ] Format MP4 720p.
- [ ] Ada intro.
- [ ] Ada subtitle.
- [ ] Demo fitur utama jelas.
- [ ] Diunggah ke YouTube.
- [ ] Judul mengikuti format LIDM.

## D. Validasi

- [ ] Survei siswa minimal 20–30 responden.
- [ ] Wawancara guru minimal 2–3 orang.
- [ ] Uji fungsionalitas.
- [ ] Uji usability.
- [ ] Pre-test dan post-test.
- [ ] Rekap hasil validasi.

## E. Presentasi

- [ ] Slide 10–12 halaman.
- [ ] Demo produk siap.
- [ ] Video backup siap.
- [ ] Internet cadangan tersedia.
- [ ] Pembagian presenter jelas.
- [ ] Latihan tanya jawab juri.
