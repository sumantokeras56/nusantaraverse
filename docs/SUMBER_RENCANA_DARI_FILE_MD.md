# Sumber Rencana Project

Project starter ini dibuat berdasarkan file rencana yang diberikan pengguna:

**Rencana_Pengerjaan_Nusantaraverse_LIDM_2026(1).md**

Poin utama yang digunakan:

1. Bentuk produk: Web App responsif + PWA.
2. Divisi: ITDP.
3. Fitur utama: peta eksplorasi, AI Tutor, misi lokasi, simulasi keputusan, gamifikasi, dashboard guru, dan eksplorasi virtual ringan.
4. Teknologi: Next.js, Tailwind CSS, AI fallback/API, dashboard analitik, dan deployment ke Vercel.
5. Output akhir: web app, proposal, video demo, validasi, slide presentasi, dan dokumen transparansi AI.
